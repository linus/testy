getTests(doc) → {Array.<TestDefinition>}

Map a doclet to a list of test definitions with their results evaluated.

Parameters:
Name Type Description
doc Array.<Doclet>

The doclet from the file under test
Source:
  • lib.js, line 54
Returns:

The test definitions parsed from the doclet, if any

Type
Array.<TestDefinition>
Example
getTests([{
  examples: ["Math.max(1, 3)\n//=> 3"],
  meta: {
    path: "/tmp",
    filename: "index.js"
  },
  name: "Math.max"
}])
//=> [
  {
    path: "/tmp/index.js",
    functionName: "Math.max",
    examples: [{
      test: "Math.max(1, 3)",
      example: "Math.max(1, 3)\n//=> 3",
      result: 3
    }]
  }
]
